# 🍔 Uber Eats Restaurant Analytics Dashboard

## 📌 Project Overview

This project presents a multi-page Power BI dashboard developed to analyze Uber Eats restaurant data across multiple continents. The dashboard provides interactive insights into restaurant performance, customer ratings, pricing, cuisine distribution, and service availability, helping support data-driven business decisions.

---

## 🎯 Project Objective

- Integrate restaurant datasets from multiple continents.
- Clean and transform data using Power Query.
- Build a star schema data model.
- Develop interactive Power BI dashboards.
- Generate business insights through DAX measures and visualizations.

---

## 📊 Dashboard Features

- Global restaurant performance overview
- Country and city drill-down analysis
- Restaurant rating analysis
- Average cost analysis
- Top-rated restaurants
- Lowest-cost restaurants
- Cuisine distribution analysis
- Online Delivery & Table Booking filters
- Interactive KPI cards and slicers

---

## 📈 DAX Measures Implemented

- Average Rating
- Average Cost for Two
- Restaurant Ranking
- Rating Classification
- Rating Range
- KPI Cards
- Conditional Formatting

---

## 💡 Key Insights

- Philippines recorded the highest average restaurant rating.
- India showed the lowest average rating, indicating improvement opportunities.
- Indonesia had the highest average cost for two.
- Australia and the United States offered comparatively lower pricing.
- Restaurants serving a wider variety of cuisines demonstrated stronger market presence.
- Online delivery adoption varied across countries.
